# Calculadora de Consumo Elétrico Inteligente
O objetivo é ajudar usuários a estimar quanto um aparelho gasta de energia elétrica por mês, com base em dados simples de uso.
## Funcionalidades
Realiza o cálculo médio mensal de energia dos eletrodomésticos
## Como Usar
Insira o nome do aparelho (ex.: “Geladeira”), a potência do aparelho em watts (W) e o tempo médio de uso diário em horas para ser feito o cálculo.
## Fórmula Utilizada
consumoMensal = (potencia * horasDia * 30) / 1000
